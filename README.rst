Quickstart
----------

Install with pip:

.. code-block:: bash

  pip install jewcal


Run in the console:

.. code-block:: bash

  jewcal


The output:

.. code-block:: console

  Today is 18 Cheshvan 5784

  JewCal(year=5784, month=8, day=18, gregorian_date=datetime.date(2023, 11, 2),
  shabbos=None, yomtov=None, category=None, diaspora=True)
